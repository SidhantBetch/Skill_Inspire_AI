# string
A string in Python is a data type used to store text or characters.  
Strings are written inside single quotes (' '), double quotes (" "), or triple quotes (''' ''').

### Example of String
    name = "Kimi"
    city = 'Rampur'
Here:
- "Kimi" is a string
- 'Rampur' is a string

### Example Program
    message = "Hello World"
    print(message)

#### Output:
    Hello World

---------------------------------------------------------------------------------------------------------------------------------------

## Accessing Characters in a String
Each character has an index number starting from 0.

### Example:
    K  i  m  i
    0  1  2  3

### Example code:
    name = "Kimi"
    print(name[0])

#### Output:
    K
---------------------------------------------------------------------------------------------------------------------------------------

## Some String Operations
### 1. String Concatenation (Joining strings)
    a = "Hello"
    b = "World"
    print(a + " " + b)
    
#### Output:
    Hello World

### 2. String Length
    name = "Kimi"
    print(len(name))

#### Output:
    4

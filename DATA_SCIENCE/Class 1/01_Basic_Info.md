# Programming
Programming means writing instructions for a computer to perform tasks using a programming language.  
A programming language helps us communicate with the computer to solve problems, create software, websites, apps, games, etc.

---------------------------------------------------------------------------------------------------------------------------------------------

## Compiler
A compiler translates the entire program at once into machine code before execution.

### Process
1. Write the program
2. Compile the program
3. If no errors → executable file is created
4. Run the program

#### Example Languages
- C
- C++
- Go

#### Example
If a C program has 10 errors, the compiler shows all errors after compilation.

### Advantages
- Program runs faster
- Errors are shown together

--------------------------------------------------------------------------------------------------------------------------------------------
 
 ## Interpreter
An interpreter translates the program line by line and executes it immediately.

### Process
1. Write the program
2. Interpreter reads one line
3. Executes that line
4. Moves to the next line

#### Example Languages
- Python
- JavaScript
- Ruby

#### Example
If there is an error on line 5, the interpreter stops there and shows the error.

### Advantages
- Easier debugging
- Good for scripting and testing

# Memory
Memory in programming is the storage area in a computer where data and programs are stored so the CPU can use them.  
When we create variables or run programs, the computer stores the data in memory.

### Example in Python
    x = 10
    name = "Kimi"
Here:
- x is stored in memory with value 10
- name is stored in memory with value "Kimi"
- So memory keeps the data of variables while the program is running.

## Types of Memory (Basic Idea)
### 1. RAM (Random Access Memory)
- Temporary memory
- Stores data while the program is running
- Data is lost when the computer is turned off

### 2. ROM (Read Only Memory)
- Permanent memory
- Stores important system instructions

## Memory Example (Simple View)
Variable    | Value
------------|----------
x           | 10
name        | Kimi

# Datatype in python
A data type in Python defines the type of value a variable can store.  
It tells the computer what kind of data is being used, such as numbers, text, or true/false values.

----------------------------------------------------------------------------------------------------------------------------------------

## Common Data Types in Python
### 1. Integer (int)
Stores whole numbers.

#### Example:
    x = 10
    y = -5

### 2. Float
Stores decimal numbers.

#### Example:
    price = 10.5
    pi = 3.14

### 3. String (str)
Stores text or characters.

#### Example:
    name = "Kimi"
    message = "Hello"

### 4. Boolean (bool)
Stores True or False values.

#### Example:
    isStudent = True
    isLoggedIn = False

### 5. List
Stores multiple values in one variable.

#### Example:
    numbers = [10, 20, 30]

### 6. Tuple
Similar to a list but cannot be changed.

#### Example:
    colors = ("red", "blue", "green")

### 7. Dictionary
Stores data in key–value pairs.

#### Example:
    student = {"name": "Kimi", "age": 20}

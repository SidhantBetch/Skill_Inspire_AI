# Python
Python is a high-level programming language that is easy to read, write, and understand. It was created by Guido van Rossum and released in 1991.  
Python is popular because its syntax is simple and similar to English, which makes it beginner-friendly.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------

## Why We Use Python
### 1. Easy to Learn
Python has simple syntax, so beginners can learn it quickly.

#### Example:
    print("Hello World")

### 2. Used in Many Fields
- Python is used in different areas such as:
- Web Development (Django, Flask)
- Data Science
- Artificial Intelligence / Machine Learning
- Automation
- Game Development
- Cybersecurity

### 3. Large Libraries
Python has many built-in libraries that make programming easier.

#### Examples:
- NumPy → mathematical calculations
- Pandas → data analysis
- TensorFlow → machine learning

### 4. Cross-Platform
- ython programs can run on:
- Windows
- Linux
- Mac

### 5. Strong Community Support
Many developers use Python, so there are lots of tutorials and help available.

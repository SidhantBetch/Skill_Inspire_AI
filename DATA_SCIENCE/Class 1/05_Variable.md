# Variable
A variable in Python is a name used to store data or a value in memory.  
The value stored in a variable can change during program execution.

### Example of Variable in Python
    x = 10
    name = "Kimi"


Here:
- x is a variable storing 10
- name is a variable storing Kimi

## Rules for Naming Variables
1. Variable must start with a letter or underscore (_)
2. It cannot start with a number
3. It cannot use Python keywords
4. Variable names are case-sensitive

### Example:
✔ Valid

    age = 20
    _name = "Ram"
    totalMarks = 90

✖ Invalid

    2age = 20
    class = 5

### Example Program
    name = "Kimi"
    age = 21
    
    print(name)
    print(age)


#### Output:
    Kimi
    21

# Type Casting
Type Casting in Python means manually converting one data type into another data type using built-in functions.  
It is done by the programmer when a specific data type is needed.

----------------------------------------------------------------------------------------------------------------------------------------------

## Common Type Casting Functions
Function	|Converts To	|Example
----------|-------------|--------
int()	|Integer	|int("10")
float()	|Float	|float(5)
str()	|String	|str(20)
bool()	|Boolean	|bool(1)

### Examples
#### 1. String to Integer
    x = "10"
    y = int(x)
    print(y)

##### Output:
    10

#### 2. Integer to Float
    a = 5
    b = float(a)
    print(b)

##### Output:
    5.0

#### 3. Number to String
    num = 25
    text = str(num)
    print(text)

##### Output:
    25

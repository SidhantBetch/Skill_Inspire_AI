# Comment
A comment in Python is a line in the program that is not executed by the interpreter.  
Comments are used to explain the code and make it easier to understand.

-------------------------------------------------------------------------------------------------------------------------------------------

## 1. Single-Line Comment
A single-line comment starts with #.

### Example
    # This is a comment
    print("Hello")
Here, the comment is ignored when the program runs.

## 2. Multi-Line Comment
Python can write multi-line comments using triple quotes.

### Example
    """
    This is a
    multi-line comment
    in Python
    """
    print("Hello")

-------------------------------------------------------------------------------------------------------------------------------------------

## Example Program with Comments
    # Store value in variable
    x = 10
    
    # Print value
    print(x)


### Output

    10

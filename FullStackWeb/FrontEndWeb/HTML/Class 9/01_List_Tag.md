# List
List tags in HTML are used to display items in a list format.  
There are three types of lists in HTML.

## 1. Ordered List (`<ol>`)
An ordered list shows items with numbers (1, 2, 3...).

### Syntax
```HTML
<ol>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ol>
```

### Example
```Html
<ol>
   <li>HTML</li>
   <li>CSS</li>
   <li>JavaScript</li>
</ol>
```

#### Output:
- HTML
- CSS
- JavaScript

## 2. Unordered List (`<ul>`)
An unordered list shows items with bullet points.

### Syntax
```html
<ul>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>
```

### Example
```html
<ul>
   <li>Apple</li>
   <li>Banana</li>
   <li>Mango</li>
</ul>
```

#### Output:
- Apple
- Banana
- Mango

## 3. Description List (`<dl>`)
A description list is used to describe terms.

### Syntax
```html
<dl>
   <dt>Term</dt>
   <dd>Description</dd>
</dl>
```

### Example
```html
<dl>
   <dt>HTML</dt>
   <dd>HyperText Markup Language</dd>

   <dt>CSS</dt>
   <dd>Cascading Style Sheets</dd>
</dl>
```

## Important Tags
Tag	|Meaning
-----|-------
`<ol>`	|Ordered list
`<ul>`	|Unordered list
`<li>`	|List item
`<dl>`	|Description list
`<dt>`	|Term
`<dd>`	|Description
